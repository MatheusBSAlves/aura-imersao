# Story 1.0: Studio Aura Landing Page

## Status
Done

## Story
Como cliente premium em Sao Paulo,
quero conhecer o Studio Aura por uma landing page sofisticada,
para avaliar o portfolio, entender o processo e iniciar contato com o estudio.

## Acceptance Criteria
- [x] Hero imersivo com video `interior-dark-video.mp4` e chamada principal em PT-BR.
- [x] Secao de metodologia com Concept, Design e Creation, incluindo video `video-camadas.mp4`.
- [x] Portfolio com pelo menos 6 projetos usando imagens de interiores.
- [x] Secao da fundadora Marina Vasconcelos com narrativa institucional.
- [x] Depoimentos com prova social de clientes.
- [x] Formulario de contato com estados de erro e sucesso.
- [x] Layout responsivo para desktop e mobile.
- [x] Implementacao feita apenas com HTML, CSS e JS.
- [x] Estilo alinhado ao `DESIGN.md` e estrutura alinhada ao PRD.

### Build Sequence Enhancement
- [x] Sequencia inicia em `start: "center center"` na secao abaixo da hero.
- [x] Area pinada ocupa 100vw e 100dvh.
- [x] Scroll controla `currentTime` sem autoplay ou loop no video da sequencia.
- [x] Video roda invertido no scroll para baixo, do final para o inicio.
- [x] Tres blocos aparecem no inicio, meio e fim, alternando esquerda, direita e esquerda.
- [x] Pin libera ao fim da timeline e reverte corretamente ao subir.
- [x] Refresh em posicao baixa sincroniza ScrollTrigger e frame do video.
